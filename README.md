# FoI-ML-DL
这是崇新学堂自己的课程，所以没有课程和书籍链接。 

主要用来记录以下常用的传统机器学习和深度学习的知识点和问题的实现方案，之后也有专门的深度学习的学习笔记。

目前是更新状态，代码会在作业上交后上传。以后就会设置为上半年可见，下半年屏蔽，以免影响学弟学妹的学习质量。
原本的目录有一些问题，现在稍微做了一些更改，但目录内的文件名没有修改。
